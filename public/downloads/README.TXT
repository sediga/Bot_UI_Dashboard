Botflows Agent (Pilot Version)
------------------------------

Thank you for testing the Botflows Agent.

This app runs locally on your Windows machine and helps automate browser workflows using a tray-based service.

INSTRUCTIONS:
1. Extract this ZIP file to any folder (e.g., Desktop or Documents).
2. Double-click `api_server.exe` to launch the agent.
3. A tray icon will appear, and the agent will start listening on http://localhost:8000
